Business requirements

  • Agent needs to be given the option to enter bank details of the person dealing with the estate if they have them. If they don’t have the bank details, they should still be able to continue through the existing journey
  • If there are arrears due, the agent should be able to pay the arrears straight away without the need to send a BR330 form
  • If the calculation can’t be done, when it is automatically run and there are arrears due, there should be no task created to send a BR330 if bank details are already held for the person dealing with the estate (DAP)
