Date of design
April 2020
Designers
Emily Wilkinson and Colin Jones

Scenario

  • Agent needs to be able to check whether a widow is entitled to inherit to any more State Pension

Business requirements

  • Agent needs to see the spouse or partner details so they can trace them on other systems
  • Agent needs to be able to change the State Pension award components if the claimant is entitled to more
  • Agent needs to be able to end the task without changing the award, if there is no further entitlement

Research notes

  • Agents were able to run through the task successfully, understanding all content on each page of the journey
  • Some agents said that inheritable components should be entered separately and then automatically update the 4 new State Pension components
